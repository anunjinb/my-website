Case study Machine learning 2025

Spot Fixes

An enterprise bug-triage engine that predicts severity at ~95% accuracy and catches duplicates — trained on 222k+ Bugzilla bug reports.

Role ML & full-stack
Timeline Fall 2025 — Spring 2026
Stack React FastAPI Supabase Random Forest
Status Open source · on GitHub

01 — Problem

The important bug is buried

Engineering teams don't lack bug reports — they drown in them. Triage is manual, the same issue gets filed five different ways, and the one critical bug sits buried under noise. Severity comes down to whoever shouts loudest, not what's actually most urgent.

The cost isn't just time. It's the S1 regression that ships because it looked like a duplicate, and the hours spent re-triaging reports a model could have scored in seconds.

02 — What I built

A model that triages for you

Spot Fixes predicts a severity level for every incoming bug, catches duplicates before they pile up, and gets sharper every time an engineer corrects it. A Random Forest classifier with TF-IDF vectorization reaches ~95% classification accuracy on 222k+ Bugzilla records, sorting reports into S1–S4 with a confidence score — while a RAG-based search engine using TF-IDF cosine similarity surfaces duplicates the moment one lands.

React on the front, FastAPI and Supabase behind it, with JWT auth, role-based access, and Row-Level Security for true multi-tenant isolation — a real product, not a notebook. The model is one piece; the platform around it is what makes the triage usable day to day.

01

Ingest at scale

222k+ Bugzilla bug records flow through an automated ETL pipeline into Supabase Postgres — the training set and the live corpus, with bulk ingestion for more.

02

Predict severity

A Random Forest classifier with TF-IDF vectorization sorts each report into S1–S4 at ~95% accuracy, attaching a confidence score.

03

Detect duplicates

A RAG-based search engine with TF-IDF cosine similarity retrieves related and duplicate reports in real time, so the same issue isn't triaged five times.

04

Learn from people

Engineer overrides are captured as ground truth and fed back into retraining, while live telemetry tracks accuracy and model drift.

03 — Outcome

Triage that earns its keep

At ~95% classification accuracy across a 222k-record corpus, Spot Fixes turns triage from a daily chore into a background process. Teams see the S1 that matters first, stop re-filing duplicates, and keep a model that learns from their own corrections rather than a fixed rulebook.

It's a full product — model, API, multi-tenant backend, and interface — built to prove that ML triage works when it's wrapped in a workflow people will actually use. Built as a Gannon senior-design project; the source is on GitHub.
